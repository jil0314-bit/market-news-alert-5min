# 무료 24시간 5분 뉴스 알림

## 무엇을 하나요?

- 형의 컴퓨터가 꺼져 있어도 GitHub가 작동합니다.
- 24시간 내내 **5분마다** 새 뉴스를 확인합니다.
- 매번 메시지를 보내는 것이 아닙니다.
- 새롭고 중요한 뉴스가 있을 때만 텔레그램으로 보냅니다.
- GitHub 예약 실행은 서버 사정에 따라 몇 분 늦어질 수 있습니다.

## 비용 없이 쓰는 조건

저장소를 **공개(Public)** 로 만드세요.
공개 저장소의 일반 GitHub Actions 실행기는 무료입니다.
코드는 공개되지만 텔레그램 토큰은 `Secrets`에 넣으므로 파일에 노출되지 않습니다.

비공개(Private) 저장소에서 5분마다 실행하면 무료 제공 시간을 넘길 가능성이 큽니다.

## 1. 새 저장소 만들기

저장소 이름 예시: `market-news-alert-5min`

공개 범위는 반드시:

`Public / 공개`

## 2. 파일 올리기

압축을 풀고 안의 파일과 폴더를 모두 업로드하세요.
특히 아래 파일이 있어야 합니다.

- `.github/workflows/market-news-alert.yml`
- `market_news_alert.py`
- `config.github.yaml`
- `requirements.txt`

## 3. 텔레그램 비밀값 넣기

저장소 → 설정 → 비밀 및 변수 → 작업 → 새 저장소 비밀

첫 번째:

- 이름: `TELEGRAM_BOT_TOKEN`
- 값: BotFather에서 새로 받은 토큰 전체

두 번째:

- 이름: `TELEGRAM_CHAT_ID`
- 값: 형의 chat_id 숫자

선택 사항:

- `NAVER_CLIENT_ID`
- `NAVER_CLIENT_SECRET`
- `DART_API_KEY`

## 4. 처음 한 번 시험

저장소 위쪽 `작업` → `글로벌 시장영향 5분 속보` → `워크플로 실행`

텔레그램 연결 성공 메시지가 오면 완료입니다.
그 뒤부터 컴퓨터를 꺼도 GitHub가 5분마다 확인합니다.

## 어떤 뉴스를 잡나요?

- 전쟁, 공격, 공습, 미사일, 제재, 호르무즈, 국제유가
- 연준, 파월, 캐빈 워시, 미국 재무부, ECB 등 정책 인사 발언
- 골드만삭스, JP모건, 모건스탠리, 블랙록 등 글로벌 IB·펀드 전망
- 젠슨 황, 일론 머스크, 사티아 나델라 등 글로벌 CEO의 투자·수요·공급·산업 전망
- 반도체, AI, 바이오, 배터리, 원전, 조선, 방산 등 한국시장 영향 뉴스
- Reuters, Financial Times, Bloomberg, CNBC, WSJ 기사는 Google News 제목·링크로 보조 확인

## 주의

이 방식은 무료형 중 가장 빠른 약 5분 확인 방식입니다.
뉴스 제공처가 실시간 웹훅을 주지 않기 때문에 기사 게시 즉시 0초 전송은 아닙니다.

채팅에 공개했던 기존 텔레그램 토큰은 반드시 폐기하고 새 토큰을 사용하세요.
